# Design: Meetup Copilot Launch

## Context
The product should feel like a live meetup utility, not a generic event site. The MVP should balance an attractive event page with clear AI moments that help a visitor make better use of the meetup.

## Decisions
- Use a single-page Node/Express app for fast shipping.
- Seed the app with realistic sample agenda items, demos, and attendee archetypes.
- Make AI user-facing via attendee matching and project idea generation.
